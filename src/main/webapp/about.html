Le projet

Ce projet se base sur une technologie Java, adaptée pour le web.
Afin d'intégrer continuellement le projet, nous utilisons GIT LAB. Ce dernier permet aussi de faire des tests JUnits.
De note côté, on effectue des tests unitaires et également des test de qualité. Ce projet est deployé sur une machine virtuelle hébergée par Google Cloud Platform.

Qui somes-nous?

Nous sommes des étudiants I4 de l'EPSI en charge d'un projet d'intégration continue pour le module "Intégration continue"
